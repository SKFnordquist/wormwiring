      • n2u_electrical.edgelist -> Electrical synaptic connections

    • wb_ws259.sql : database with gene expression data from Wormbase version WS259
    • cam_nr_pre_post.csv : list of CAM genes used in the analysis
      Row format: gene name, presynaptic expressed, postsynaptic expressed, number of known isoforms
    • cluster_networks.cys : Cytoscape file for displaying expression clusters
    • exp_cluster.gml : Graph file for expression clusters under the Whole-cell Binary Expression model (WBE)

      Row format: cell name, presynaptic LUS, gap junction LUS
    • N2U_sbe.csv : Adult LUS scores for the sub-cellular binary expression model (SBE). Computed using run_cam_models.py
      Row format: cell name, presynaptic LUS, gap junction LUS
    • N2U_ie.csv : Adult LUS scores for 1000 iterations of IE model. LUS scores are the average across all cells. Computed using run_cam_models.py
      Row format: average presynaptic LUS score, average gap_junction LUS score.

    • JSH_sbe.csv : L4 LUS scores for the sub-cellular binary expression model (SBE). Computed using run_cam_models.py

    • JSH_ie.csv : L4 LUS scores for 1000 iterations of IE model. LUS scores are the average across all cells. Computed using run_cam_models.py
      Row format: average presynaptic LUS score, average gap_junction LUS score.
    • aiy_lus.gml : Graph file for LUS score of cell AIY.
